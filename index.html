gh.mattrude.com / The Kirby Theme

Kirby is a WordPress Child Theme of the Twenty Eleven theme, bundled within WordPress 3.2 & WordPress 3.3.

The Kirby Child Theme adds the following features to Twenty Eleven.

  • EXIF Support for images
  • Sidebar on Single Pages and Posts
  • Wider screen display
  • Yet Another Related Posts Plugin support
  • Other minor tweeks to the CSS

Prerequisites

  • WordPress 3.2+
  • Twenty Eleven Theme 1.2+

Support

If you are having issues using the Kirby Theme, please file a bug on this projects github Issues page.
